Για κάθε μέρος, το testbench έχει σχεδιαστεί να δουλεύει με το αργό ρολόι.
Για να τρέξει το simulation και να φανούν τα σήματα γρήγορα χρειάζεται το εξής:
-Στο μέρος Β και Γ, στο debounce module η παράμετρος DEBOUNCE_DELAY να έχει τιμή 3.

-Στο μέρος Δ, στο debounce module η παράμετρος DEBOUNCE_DELAY να έχει τιμή 3
και στο memory module η παράμετρος BIT_LENGTH = 5.
